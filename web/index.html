Most RAG systems answer every question — even when the documents don't contain the answer. That confident, fluent, wrong answer is a hallucination. VeriRAG grades its own evidence and abstains when it isn't sure.

No login needed · runs entirely on free / local models · ₹0 API cost

✗ Conventional RAG
  • 🎯Retrieves top-k chunks whether or not anything relevant exists.
  • 💬Hands them to a model trained above all to be helpful.
  • 🔥Returns a confident, cited answer — even when it's invented.
  • ⚠️A wrong answer is indistinguishable from a correct one.
✓ VeriRAG
  • ⚖️Grades retrieved evidence before it writes anything.
  • 🔬Verifies its own answer against the sources afterward.
  • 🔁Re-queries, asks a clarifying question, or abstains when unsure.
  • 🛡️Every answer ships with a confidence score and a decision trace.

The agent loop

Two-stage anti-hallucination

A grader runs before generation to catch bad retrieval. A verifier runs after to catch invention. Either alone leaves a hole.
